Automate JFORM Creation using JiraRestClient

Designed and implemented a novel REST API endpoint for Cerner's Jira instance queue (jira.cerner.com), which allows users to input data in four distinct sections: General, Design Input, Design Output, and Testcases. Utilized the JiraRestClient library to execute a POST request with the provided inputs, resulting in the creation of new JFORMs on the server. To streamline the process, automation was employed to handle the request, capturing all input fields and displaying them in Swagger UI for user review. Adopting a Test Driven Development approach, each module underwent rigorous testing using Junit test cases and the Mockito framework, resulting in an impressive code coverage of 95%. Managed the codebase using GIT and ensured continuous integration through Jenkins.

Automated Dynamic Detection of Ransomware using Augmented Bootstrapping (Bachelor's Thesis)

This study employs a systematic approach to classify samples as either ransomware or benign, with a primary focus on dynamic analysis techniques for feature extraction from executable files. NLP-based language modeling techniques, such as N-gram construction and TF-IDF scores, are utilized to extract the most significant features from the samples. A cluster of classification models, including Gaussian Naïve Bayes, Logistic Regression, SVM, Random Forest, and KNN, are trained to effectively identify ransomware. To enhance detection accuracy, ensemble learning-based detection solutions (Hard Voting Classifier) are proposed, combining multiple weak classifiers.

Zomato Data Analysis

The main purpose behind analyzing the Zomato dataset is to gain valuable insights into the various factors that influence the establishment of different types of restaurants across various countries and cities. Additionally, the analysis aims to identify similarities between neighborhoods in terms of food preferences. The data used for this study has been obtained from Kaggle.com in the form of raw .csv files.
